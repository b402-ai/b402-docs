This page is optimized for AI agents and LLMs. It provides a concise, structured reference for every SDK method with exact parameters, return types, and example calls.

## Setup

```bash
npm install @b402ai/sdk
```

```typescript
import { B402 } from '@b402ai/sdk'

const b402 = new B402({
  privateKey: process.env.PRIVATE_KEY,       // required
  zeroXApiKey: process.env.ZERO_X_API_KEY,   // optional, for swaps
})
```

Only `privateKey` is required. The b402 facilitator handles gas, wallet deployment, and transaction submission.

## Commands

### Shield (Enter Privacy Pool)

Move tokens from EOA into the Railgun privacy pool. Breaks the on-chain link between your real wallet and smart wallet.

```typescript
const result = await b402.shield({ token: 'USDC', amount: '100' })
// { txHash: '0x...', indexed: true }
```

| Param | Required | Type | Values |
|-------|----------|------|--------|
| `token` | Yes | `string` | `USDC`, `WETH`, `DAI` |
| `amount` | Yes | `string` | Human-readable (e.g. `'100'`) |

Shield sends from master EOA — requires tokens + ETH for gas. Indexing takes 1-3 min (SDK polls automatically).

### Unshield (Exit Privacy Pool)

Withdraw tokens from the privacy pool to your anonymous smart wallet. Generates a ZK proof (Groth16) client-side.

```typescript
const result = await b402.unshield({ token: 'USDC', amount: '50' })
// { txHash: '0x...', proofTimeSeconds: 0.7 }
```

| Param | Required | Type | Values |
|-------|----------|------|--------|
| `token` | Yes | `string` | `USDC`, `WETH`, `DAI` |
| `amount` | Yes | `string` | Human-readable |

b402 uses a Railgun fork with 0% protocol fees — no fee is deducted on unshield.

### Swap (Private Token Swap)

Swap tokens via 0x aggregator. Requires `zeroXApiKey` in config.

```typescript
const result = await b402.swap({
  from: 'USDC',
  to: 'WETH',
  amount: '10',
  slippageBps: 100,
})
// { txHash: '0x...', amountIn: '10', amountOut: '0.004', tokenIn: 'USDC', tokenOut: 'WETH' }
```

| Param | Required | Type | Values |
|-------|----------|------|--------|
| `from` | Yes | `string` | Token symbol or `0x` address |
| `to` | Yes | `string` | Token symbol or `0x` address |
| `amount` | Yes | `string` | Human-readable sell amount |
| `slippageBps` | No | `number` | Basis points (default: `100` = 1%) |

### Lend (Deposit into Morpho Vault)

```typescript
const result = await b402.lend({ token: 'USDC', amount: '100', vault: 'steakhouse' })
// { txHash: '0x...', amount: '100', vault: 'steakhouse' }
```

| Param | Required | Type | Values |
|-------|----------|------|--------|
| `token` | Yes | `string` | `USDC` |
| `amount` | Yes | `string` | Human-readable |
| `vault` | No | `string` | `steakhouse` (default), `moonwell`, `gauntlet`, `steakhouse-hy` |

### Redeem (Withdraw from Vault)

```typescript
const result = await b402.redeem({ vault: 'steakhouse' })
// { txHash: '0x...', assetsReceived: '100.23', vault: 'steakhouse' }
```

| Param | Required | Type | Values |
|-------|----------|------|--------|
| `vault` | No | `string` | Default: `steakhouse` |
| `shares` | No | `string` | Shares to redeem. Default: all |

### Rebalance (Auto-Optimize Yield)

```typescript
const result = await b402.rebalance(0.5) // min 0.5% APY difference to trigger
// { action: 'rebalanced', currentVault: 'steakhouse', bestVault: 'steakhouse-hy', txHash: '0x...' }
```

### Transact (Arbitrary Calls)

Execute any batch of EVM calls through your smart wallet. Gasless.

```typescript
const result = await b402.transact([
  { to: '0xTokenAddr', value: '0', data: '0xapproveCalldata' },
  { to: '0xProtocol',  value: '0', data: '0xactionCalldata' },
])
// { txHash: '0x...' }
```

### Status (Check Wallet)

```typescript
const status = await b402.status()
// {
//   ownerEOA: '0x...',
//   smartWallet: '0x...',
//   deployed: true,
//   chain: 'base',
//   balances: [{ token: 'USDC', balance: '50.0' }],
//   positions: [{ vault: 'steakhouse', shares: '49.5', assets: '100.5 USDC', apyEstimate: '4-6%' }]
// }
```

## Error Reference

| Error | Cause | Fix |
|-------|-------|-----|
| `privateKey is required` | Missing private key | Pass `privateKey` in constructor |
| `zeroXApiKey required for swaps` | No 0x key | Pass `zeroXApiKey` in constructor |
| `Unknown token: X` | Invalid token symbol | Use `USDC`, `WETH`, or `DAI` |
| `Unknown vault: X` | Invalid vault name | Use `steakhouse`, `moonwell`, `gauntlet`, `steakhouse-hy` |
| `No shares in X` | No position in vault | Deposit first with `b402.lend()` |
| `No shielded balance for X` | No UTXOs | Shield tokens first with `b402.shield()` |
| `Insufficient shielded balance` | UTXO too small | Shield more tokens |
| `Facilitator verify failed` | UserOp rejected | Check wallet has tokens, retry |
