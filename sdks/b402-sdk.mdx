## Install

```bash
npm install @b402ai/sdk
```

<Info>
  Requires Node.js 18+. The SDK uses snarkjs for client-side ZK proof generation.
</Info>

## Quick Start

```typescript
import { B402 } from '@b402ai/sdk'

const b402 = new B402({
  privateKey: process.env.PRIVATE_KEY,
})
```

One key. The b402 facilitator handles gas, wallet deployment, and UserOp submission.

<Note>
  Your `privateKey` derives a deterministic **incognito wallet** — cryptographically unlinkable to any other address. The facilitator sponsors gas via paymaster. You pay $0 for gas.
</Note>

## The Privacy Flow

Tokens enter the privacy pool, then exit untraceable to your smart wallet.

```
EOA → [shield] → Privacy Pool → [unshield] → Smart Wallet → [DeFi]
         ↑                           ↑                          ↑
    One-time setup           ZK proof breaks          Gasless via
    (needs gas)              the funding link          facilitator
```

### Step 1: Shield — Enter the Privacy Pool

Move tokens from your EOA into the Railgun privacy pool. After this, nobody can trace where the tokens came from.

```typescript
const shield = await b402.shield({ token: 'USDC', amount: '100' })
console.log(shield.txHash) // 0x...
// Tokens are now in the privacy pool — untraceable
```

### Step 2: Unshield — Exit to Smart Wallet

Pull tokens from the privacy pool to your anonymous smart wallet. A ZK proof is generated client-side (Groth16) — proving you own the tokens without revealing the deposit.

```typescript
const unshield = await b402.unshield({ token: 'USDC', amount: '50' })
console.log(unshield.txHash)           // 0x...
console.log(unshield.proofTimeSeconds) // 0.7
// USDC is now on your smart wallet, funding source hidden
```

### Step 3: DeFi — Swap, Lend, Yield

Execute DeFi operations from the smart wallet. All gasless via facilitator.

```typescript
// Swap USDC → WETH
const swap = await b402.swap({ from: 'USDC', to: 'WETH', amount: '10' })

// Earn yield anonymously in Morpho vault
await b402.lend({ token: 'USDC', amount: '100', vault: 'steakhouse' })

// Withdraw
const redeem = await b402.redeem({ vault: 'steakhouse' })
```

### Step 4: Execute Anything

Run arbitrary calls through your smart wallet — any contract, any function.

```typescript
const result = await b402.transact([
  { to: '0xTokenAddr', value: '0', data: '0xapproveCalldata...' },
  { to: '0xProtocol',  value: '0', data: '0xactionCalldata...' },
])
console.log(result.txHash)
```

## Full API

### `shield(params)` — Enter Privacy Pool

Deposit tokens from your EOA into the Railgun privacy pool. This is the onboarding step — it breaks the on-chain link between your real wallet and smart wallet.

```typescript
const result = await b402.shield({ token: 'USDC', amount: '100' })
```

| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `token` | `string` | Yes | Token symbol (`USDC`, `WETH`, or `DAI` on Base) |
| `amount` | `string` | Yes | Human-readable amount to shield |

**Returns:** `{ txHash: string, indexed: boolean }`

<Info>
  Shield sends a transaction from your master EOA — requires the EOA to hold tokens + ETH for gas. The SDK waits for the backend to index the shielded UTXO (polls every 5s, up to 3 minutes) before returning.
</Info>

---

### `unshield(params)` — Exit Privacy Pool

Withdraw tokens from the privacy pool to your anonymous smart wallet. Generates a ZK proof client-side (Groth16) proving ownership without revealing the deposit.

The SDK automatically handles partial unshields — if the UTXO value exceeds the requested amount, it creates a change note back to your Railgun address.

```typescript
const result = await b402.unshield({ token: 'USDC', amount: '50' })
```

| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `token` | `string` | Yes | Token to unshield (`USDC`, `WETH`, `DAI`) |
| `amount` | `string` | Yes | Human-readable amount to unshield |

**Returns:** `{ txHash: string, proofTimeSeconds: number }`

<Info>
  ZK proof generation happens entirely client-side — your private key never leaves your machine. Proof time depends on hardware.
</Info>

### `transact(calls)` — Execute Arbitrary Calls

Send any batch of calls through your smart wallet via the facilitator. Gasless — paymaster sponsors gas.

```typescript
import { ethers } from 'ethers'

const erc20 = new ethers.Interface(['function transfer(address to, uint256 amount)'])

const result = await b402.transact([
  {
    to: '0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913', // USDC
    value: '0',
    data: erc20.encodeFunctionData('transfer', ['0xRecipient', ethers.parseUnits('10', 6)]),
  },
])
console.log(result.txHash)
```

| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `calls` | `Call[]` | Yes | Array of `{ to: string, value: string, data: string }` |

**Returns:** `{ txHash: string }`

This is the low-level primitive. `swap()`, `lend()`, and `redeem()` are built on top of `transact()`.

---

### `swap(params)` — Private Token Swap

Swap tokens via 0x aggregator. Requires `zeroXApiKey` in the constructor.

```typescript
const b402 = new B402({
  privateKey: '0x...',
  zeroXApiKey: process.env.ZERO_X_API_KEY,
})

const result = await b402.swap({
  from: 'USDC',
  to: 'WETH',
  amount: '10',
  slippageBps: 100, // 1% default
})
```

| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `from` | `string` | Yes | Sell token symbol or address |
| `to` | `string` | Yes | Buy token symbol or address |
| `amount` | `string` | Yes | Human-readable sell amount |
| `slippageBps` | `number` | No | Slippage in bps (default: 100 = 1%) |

**Returns:** `{ txHash: string, amountIn: string, amountOut: string, tokenIn: string, tokenOut: string }`

The SDK builds an `approve` + `swap` call pair and executes both atomically via the facilitator.

### `redeem(params)` — Withdraw from Vault

```typescript
const result = await b402.redeem({ vault: 'steakhouse' })
console.log(result.assetsReceived) // '100.23' USDC
```

| Param | Type | Required | Description |
|-------|------|----------|-------------|
| `vault` | `string` | No | Vault name (default: `steakhouse`) |
| `shares` | `string` | No | Shares to redeem (default: all) |

**Returns:** `{ txHash: string, assetsReceived: string, vault: string }`

---

### `rebalance(minApyDiff?)` — Auto-Rebalance

Move capital to the highest-yield vault. Compares your current vault position APY against all available vaults, and rebalances if the difference exceeds the threshold.

```typescript
const result = await b402.rebalance(0.5) // min 0.5% APY diff to trigger
```

**Returns:** `{ action: 'rebalanced' | 'no-change', currentVault?: string, bestVault?: string, txHash?: string }`

const status = await b402.status()

Reads balances for all supported tokens (USDC, WETH, DAI) and positions across all Morpho vaults in parallel.

## How It Works

### Wallet Derivation

```
privateKey
  → sign("b402 Incognito EOA Derivation")
  → keccak256(signature)
  → incognitoPrivateKey
  → Wallet(incognitoPrivateKey)
  → incognitoEOA (deterministic, unlinkable)

incognitoEOA
  → salt = keccak256("b402-incognito-" + eoa.toLowerCase())
  → NexusFactory.computeAccountAddress(initData, salt)
  → smartWallet (CREATE2, deployed on first use)
```

### Execution Flow (Facilitator)

```
SDK builds calls[] → POST /api/v1/wallet/incognito/verify
  → facilitator builds UserOp + paymaster sig
  → SDK signs userOpHash with incognito wallet
  → POST /api/v1/wallet/incognito/settle
  → facilitator submits to bundler → relayer pays gas
```

### ZK Proof Flow (Shield / Unshield)

```
Shield:    EOA → approve → populateShield() → send tx → wait for indexing
Unshield:  deriveKeys → fetchUTXOs → buildProofInputs → generateProof → buildUnshieldTx
```

Proofs are Groth16 (snarkjs), generated entirely client-side. Your private key never leaves your machine.

### What's Visible vs Hidden

| Visible On-Chain | Hidden |
|------------------|--------|
| Smart wallet address | Who owns it |
| DeFi action + amount | Funding source (privacy pool) |
| Block number | Operator's real EOA |
| Paymaster sponsorship | Link between shield and unshield |

## Progress Tracking

```typescript
const b402 = new B402({
  privateKey: '0x...',
  onProgress: (event) => {
    if (event.type === 'step') console.log(`[${event.step}/${event.totalSteps}] ${event.title}`)
    else if (event.type === 'done') console.log(`Done: ${event.message}`)
    else if (event.type === 'info') console.log(`${event.title}: ${event.message}`)
  },
})
```

Progress events:

| Type | Fields | When |
|------|--------|------|
| `step` | `step`, `totalSteps`, `title`, `message` | Each execution step (build UserOp, sign, submit) |
| `done` | `title`, `message` | Transaction confirmed |
| `info` | `title`, `message` | Informational updates (proof time, wallet deployment) |

Gas for all DeFi operations is sponsored by the facilitator. The standard b402 protocol fees apply to payments — see [Fee Structure](/concepts/fees) for details.

| Fee | Amount |
|-----|--------|
| Gas (DeFi ops) | $0.00 (facilitator-sponsored) |

| Swap | DEX fee (varies by pair) |

## Error Handling

```typescript
try {
  await b402.swap({ from: 'USDC', to: 'WETH', amount: '10' })
} catch (err) {
  console.error('Failed:', err.message)
}
```

| Error | Fix |
|-------|-----|
| `privateKey is required` | Pass `privateKey` in constructor |
| `zeroXApiKey required for swaps` | Pass `zeroXApiKey` in constructor |
| `Unknown token: X` | Use supported tokens: `USDC`, `WETH`, `DAI` — or pass a valid `0x` address |
| `Unknown vault: X` | Use: `steakhouse`, `moonwell`, `gauntlet`, `steakhouse-hy` |
| `No shares in X` | No position — deposit first with `lend()` |
| `calls array is required and must not be empty` | Pass at least one call to `transact()` |
| `No shielded balance for X. Shield tokens first` | No UTXOs found — run `shield()` first |
| `Insufficient shielded balance` | Shield more tokens |
| `Facilitator verify failed` | Check facilitator health, try again |
| `UserOp rejected` | Invalid calls or insufficient wallet balance |

## For Agents

Every method takes plain strings and returns plain objects. No ABI encoding, no gas estimation, no nonce management.

```typescript

await b402.lend({ token: 'USDC', amount: '1000', vault: 'steakhouse' })
```
