The `b402-sdk` is a TypeScript SDK that handles EIP-712 signature construction, facilitator communication, and on-chain settlement for b402 payments. It abstracts the full payment flow into a single method call.

## Installation

```bash
npm install b402-sdk ethers
```

Requires Node.js 18+ and ethers.js v6.

## Quick Start

```typescript
import { Wallet, JsonRpcProvider } from 'ethers';
import { B402 } from 'b402-sdk';

const provider = new JsonRpcProvider('https://bsc-dataseed1.binance.org');
const wallet = new Wallet(process.env.PRIVATE_KEY, provider);

const b402 = new B402('mainnet');

const result = await b402.pay(wallet, {
  amount: '0.01',
  token: 'USD1',
  recipient: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
});

if (result.success) {
  console.log('TX:', result.txHash);
}
```

## `B402` Class

### Constructor

```typescript
const b402 = new B402(network: 'mainnet' | 'testnet');
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `network` | `'mainnet' \| 'testnet'` | Target network. `'mainnet'` uses BNB Chain (chainId 56). `'testnet'` uses BNB Chain Testnet (chainId 97). |

The constructor configures the facilitator URL, relayer contract address, and supported token addresses based on the selected network.

## Core Methods

### `b402.pay(signer, options)`

Signs an EIP-712 payment authorization and submits it to the b402 Facilitator for on-chain settlement. The Facilitator covers gas fees, so the payer only needs token balance and a one-time approval.

```typescript
const result = await b402.pay(signer, {
  amount: '1.50',
  token: 'USDT',
  recipient: '0xSellerAddress...',
});
```

**Parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `signer` | `ethers.Wallet` | An ethers.js Wallet instance connected to a provider. Used to sign the EIP-712 authorization. |
| `options` | `PaymentOptions` | Payment configuration (see below). |

**`PaymentOptions` Interface**

```typescript
interface PaymentOptions {
  amount: string;      // Human-readable token amount (e.g., '0.01', '100')
  token: 'USDT' | 'USDC' | 'USD1';  // Token symbol
  recipient: string;   // Seller's wallet address (checksummed)
}
```

| Field | Type | Description |
|-------|------|-------------|
| `amount` | `string` | Payment amount in token units (not wei). For example, `'1.50'` means 1.50 USDT. The SDK converts to the correct on-chain precision (18 decimals on BSC). |
| `token` | `'USDT' \| 'USDC' \| 'USD1'` | Token symbol. Must be a supported stablecoin on the selected network. |
| `recipient` | `string` | The seller's wallet address that will receive the payment. |

**Returns: `Promise<PaymentResult>`**

```typescript
interface PaymentResult {
  success: boolean;    // Whether the payment settled on-chain
  txHash?: string;     // Transaction hash (present on success)
  payer?: string;      // Payer's wallet address
  network?: string;    // Network the payment settled on
  blockNumber?: number; // Block number of the settlement transaction
  error?: string;      // Error message (present on failure)
}
```

**Under the hood**, `b402.pay()` performs these steps:

1. Resolves the token contract address from the symbol
2. Generates a random 32-byte nonce
3. Sets `validAfter` to `0` and `validBefore` to current time + 1 hour
4. Constructs the EIP-712 `TransferWithAuthorization` message
5. Signs with `wallet.signTypedData()`
6. Sends the payload to `POST /api/v1/verify` on the Facilitator
7. If valid, sends to `POST /api/v1/settle` for on-chain execution
8. Returns the settlement result

### `b402.getBalance(address, token)`

Reads the on-chain token balance for a given wallet address.

```typescript
const balance = await b402.getBalance(wallet.address, 'USD1');
console.log('Balance:', balance); // e.g., '142.50'
```

**Parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `address` | `string` | Wallet address to check. |
| `token` | `'USDT' \| 'USDC' \| 'USD1'` | Token symbol. |

**Returns: `Promise<string>`** - The balance in human-readable token units.

---

### `b402.approve(signer, token)`

Sends an on-chain approval transaction allowing the b402 relayer contract to transfer tokens on behalf of the signer. This is a one-time operation per token. Unlike `b402.pay()`, this method does require a small amount of BNB for gas.

```typescript
const tx = await b402.approve(wallet, 'USD1');
console.log('Approval TX:', tx.hash);
await tx.wait(); // Wait for confirmation
```

**Parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `signer` | `ethers.Wallet` | Wallet instance connected to a provider. Must have BNB for gas. |
| `token` | `'USDT' \| 'USDC' \| 'USD1'` | Token symbol to approve. |

**Returns: `Promise<ethers.TransactionResponse>`** - The approval transaction. Call `.wait()` to confirm.

<Note>
  Approval is required once per token before calling `b402.pay()`. If you skip this step and the relayer has no allowance, `b402.pay()` will fail with an `INSUFFICIENT_ALLOWANCE` error.
</Note>

---

### `b402.getAllowance(address, token)`

Checks the current token allowance granted to the b402 relayer contract by a given wallet.

```typescript
const allowance = await b402.getAllowance(wallet.address, 'USDT');

if (allowance === '0') {
  await b402.approve(wallet, 'USDT');
}
```

**Parameters**

| Parameter | Type | Description |
|-----------|------|-------------|
| `address` | `string` | Wallet address to check. |
| `token` | `'USDT' \| 'USDC' \| 'USD1'` | Token symbol. |

**Returns: `Promise<string>`** - The current allowance in human-readable token units.

## Error Handling

The SDK throws typed errors for common failure scenarios. Wrap calls in try/catch to handle them:

```typescript
import { B402, B402Error } from 'b402-sdk';

try {
  const result = await b402.pay(wallet, options);

  if (!result.success) {
    console.error('Settlement failed:', result.error);
  }
} catch (error) {
  if (error instanceof B402Error) {
    switch (error.code) {
      case 'INSUFFICIENT_BALANCE':
        console.error('Not enough tokens. Check balance first.');
        break;
      case 'INSUFFICIENT_ALLOWANCE':
        console.error('Token not approved. Call b402.approve() first.');
        break;
      case 'INVALID_SIGNATURE':
        console.error('Signature verification failed.');
        break;
      case 'FACILITATOR_ERROR':
        console.error('Facilitator service error:', error.message);
        break;
      default:
        console.error('Payment error:', error.message);
    }
  }
}
```

| Error Code | Description |
|------------|-------------|
| `INSUFFICIENT_BALANCE` | Wallet does not have enough tokens for the payment amount. |
| `INSUFFICIENT_ALLOWANCE` | The relayer contract has not been approved to spend tokens. Call `b402.approve()`. |
| `INVALID_SIGNATURE` | The EIP-712 signature failed verification on the Facilitator. |
| `FACILITATOR_ERROR` | The Facilitator API returned an error (network issue, settlement failure). |
| `INVALID_TOKEN` | The token symbol is not supported on the selected network. |
| `INVALID_NETWORK` | The network parameter is not `'mainnet'` or `'testnet'`. |

## Full End-to-End Example

This example covers the complete buyer flow: checking balance, approving the relayer, and sending a payment.

```typescript
import { Wallet, JsonRpcProvider } from 'ethers';
import { B402, B402Error } from 'b402-sdk';

const RPC = 'https://bsc-dataseed1.binance.org';

async function payForService() {
  // 1. Initialize provider and wallet
  const provider = new JsonRpcProvider(RPC);
  const wallet = new Wallet(process.env.PRIVATE_KEY, provider);

  console.log('Wallet:', wallet.address);

  // 2. Initialize SDK
  const b402 = new B402('mainnet');

  // 3. Check token balance
  const balance = await b402.getBalance(wallet.address, 'USD1');
  console.log('USD1 Balance:', balance);

  if (parseFloat(balance) < 0.01) {
    console.error('Insufficient USD1 balance');
    process.exit(1);
  }

  // 4. Ensure relayer is approved (one-time per token)
  const allowance = await b402.getAllowance(wallet.address, 'USD1');

  if (allowance === '0') {
    console.log('Approving relayer for USD1...');
    const approveTx = await b402.approve(wallet, 'USD1');
    await approveTx.wait();
    console.log('Relayer approved');
  }

  // 5. Send payment
  try {
    const result = await b402.pay(wallet, {
      amount: '0.01',
      token: 'USD1',
      recipient: '0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb',
    });

    if (result.success) {
      console.log('Payment settled on-chain');
      console.log('TX Hash:', result.txHash);
      console.log('Block:', result.blockNumber);
      console.log(`https://bscscan.com/tx/${result.txHash}`);
    } else {
      console.error('Settlement failed:', result.error);
    }
  } catch (error) {
    if (error instanceof B402Error) {
      console.error(`[${error.code}] ${error.message}`);
    } else {
      throw error;
    }
  }
}

payForService();
```
