## Network Support

**b402** verifies and settles payments on EVM chains, with BNB Chain and Base as the primary execution networks for gasless and agent-native transactions.  

Facilitators process and confirm on-chain settlements through B402Relayer contracts on each supported network, ensuring secure and verifiable payment flows.


## Token Support

**b402** currently supports the following assets on the BNB Chain:

- **BNB**
- **USDT** 
- **USD1**

Additional BEP-20 token support on BNB Chain, as well as assets on other EVM networks like Base, will be added as the protocol evolves.
