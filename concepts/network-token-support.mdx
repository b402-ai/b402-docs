b402 operates across EVM-compatible chains with support for multiple stablecoins. All payments are settled on-chain through the b402 Relayer contract, which is deployed on each supported network.

## Token Notes

**Token decimals vary by chain.** On BNB Chain, all tokens use 18 decimals (1 USDT = `1000000000000000000`). On Base, USDC uses 6 decimals (1 USDC = `1000000`). Always check the token's decimals for the target chain.

**Token whitelisting.** Only tokens whitelisted on the Relayer contract can be used for payments. The tokens listed above are currently whitelisted.

**Adding new tokens.** New token support requires whitelisting on the Relayer contract by the contract owner. Contact the b402 team to request new token support.
