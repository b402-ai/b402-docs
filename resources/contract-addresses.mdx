## Notes

- **Token decimals vary by chain.** BNB Chain tokens use 18 decimals. Base USDC uses 6 decimals. Always check decimals for the target chain.
- **Token whitelisting** — Only tokens whitelisted on the Relayer contract can be used for payments.
- See [Network & Token Support](/concepts/network-token-support) for detailed token and chain information.
