## Quick Overview

### For Buyers (30 seconds)

```bash
npm install ethers b402-sdk
```

```typescript
import { Wallet, JsonRpcProvider } from 'ethers';
import { B402 } from 'b402-sdk';

const provider = new JsonRpcProvider('https://bsc-dataseed1.binance.org');
const wallet = new Wallet(process.env.PRIVATE_KEY, provider);
const b402 = new B402('mainnet');

const result = await b402.pay(wallet, {
  amount: '0.01',
  token: 'USD1',
  recipient: '0x...sellerAddress',
});

console.log('TX:', result.txHash);
```

<Warning>
Before your first payment, you must approve the b402 Relayer to spend your tokens. See the [Buyer Guide](/get-started/buyer) for the one-time approval step.
</Warning>

### For Sellers (30 seconds)

```bash
npm install express b402-express
```

```typescript
const express = require('express');
const { b402 } = require('b402-express');

const app = express();

app.use(b402(
  { payTo: '0x...yourWallet' },
  { 'GET /premium': { price: '0.01', token: 'USD1' } }
));

app.get('/premium', (req, res) => {
  res.json({ data: 'paid content', payer: req.b402.payer });
});

app.listen(3000);
```

No API keys needed. The middleware handles `402` responses, payment verification, and settlement automatically.
