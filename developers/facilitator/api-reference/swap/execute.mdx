---
title: 'Execute Swap'
description: 'Execute a token swap with EOA payment'
---

`POST /swap` executes a token swap on PancakeSwap V3 using an EIP-712 signed payment. The facilitator verifies the payment, executes the swap, settles the payment, and returns the results in a single call.

```
POST /swap
```

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `paymentPayload` | `object` | Yes | EIP-712 signed payment data (same format as [`/verify`](/developers/facilitator/api-reference/eoa-payments/verify)) |
| `swapParams` | `object` | Yes | Swap configuration |
| `swapParams.tokenIn` | `string` | Yes | Input token contract address |
| `swapParams.tokenOut` | `string` | Yes | Output token contract address |
| `swapParams.amountIn` | `string` | Yes | Input amount in wei |
| `swapParams.minAmountOut` | `string` | Yes | Minimum acceptable output in wei (slippage protection) |
| `swapParams.deadline` | `number` | Yes | Unix timestamp after which the swap expires |

## Response

### 200 - Swap Executed

| Field | Type | Description |
|-------|------|-------------|
| `success` | `boolean` | Whether the swap completed successfully |
| `txHash` | `string` | Payment settlement transaction hash |
| `amountIn` | `string` | Actual input amount in wei |
| `amountOut` | `string` | Actual output amount in wei |
| `swapTxHash` | `string` | PancakeSwap swap transaction hash |

### Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 400 | `validation_error` | Missing or invalid fields |
| 400 | `signature_error` | EIP-712 signature is malformed or recovery failed |
| 400 | `swap_error` | Swap failed (slippage exceeded, deadline passed, insufficient liquidity) |
| 500 | `blockchain_error` | On-chain execution or RPC failure |

## Execution Flow

1. **Verify payment** - Validates the EIP-712 signature and checks on-chain state
2. **Execute swap** - Submits the swap to PancakeSwap V3 with the specified parameters
3. **Settle payment** - Executes `transferWithAuthorization()` on the Relayer contract
4. **Return results** - Returns both the swap and payment transaction details

<Warning>
Set `minAmountOut` to protect against slippage. If the output falls below this amount, the swap reverts. Use [`GET /swap/quote`](/developers/facilitator/api-reference/swap/quote) to determine a reasonable value.
</Warning>

```bash
  -d '{
    "paymentPayload": {
      "token": "0x55d398326f99059fF775485246999027B3197955",
      "payload": {
        "authorization": {
          "from": "0xAbC1234567890aBcDeF1234567890AbCdEf123456",
          "to": "0x9876543210FeDcBa9876543210FeDcBa98765432",
          "value": "10000000000000000000",
          "validAfter": 1709251200,
          "validBefore": 1709254800,

        },
        "signature": "0xabc123...eip712_signature"
      }

    "swapParams": {

  }'
```

### TypeScript

```typescript
// 1. Get a quote to determine minAmountOut
const quoteParams = new URLSearchParams({
  tokenIn: '0x55d398326f99059fF775485246999027B3197955',
  tokenOut: '0xbb4CdB9CBd36B01bD1cBaEBF2De08d9173bc095c',
  amountIn: '10000000000000000000',
});
const quoteRes = await fetch(
  `https://facilitator.b402.ai/swap/quote?${quoteParams}`
);
const quote = await quoteRes.json();

// 2. Apply 1% slippage tolerance
const minAmountOut = BigInt(quote.amountOut) * 99n / 100n;

// 3. Execute the swap
const response = await fetch('https://facilitator.b402.ai/swap', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    paymentPayload: {
      token: '0x55d398326f99059fF775485246999027B3197955',
      payload: {
        authorization: {
          from: wallet.address,
          to: recipientAddress,
          value: '10000000000000000000',
          validAfter: Math.floor(Date.now() / 1000),
          validBefore: Math.floor(Date.now() / 1000) + 3600,
          nonce: crypto.randomBytes(32).toString('hex'),
        },
        signature: eip712Signature,
      },
    },
    swapParams: {
      tokenIn: '0x55d398326f99059fF775485246999027B3197955',
      tokenOut: '0xbb4CdB9CBd36B01bD1cBaEBF2De08d9173bc095c',
      amountIn: '10000000000000000000',
      minAmountOut: minAmountOut.toString(),
      deadline: Math.floor(Date.now() / 1000) + 3600,
    },
  }),
});

const result = await response.json();

if (result.success) {
  console.log('Swap executed');
  console.log('Amount in:', result.amountIn);
  console.log('Amount out:', result.amountOut);
  console.log('Swap TX:', result.swapTxHash);
  console.log('Payment TX:', result.txHash);
} else {
  console.error('Swap failed');
}
```

## Notes

- PancakeSwap V3 integration on BNB Chain
- Supports any token pair with liquidity on PancakeSwap V3
- The `deadline` parameter prevents stale swaps from executing
- The facilitator covers gas costs for the swap transaction
