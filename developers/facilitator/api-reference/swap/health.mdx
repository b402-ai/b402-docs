---
title: 'Swap Health Check'
description: 'Check swap service availability'
---

`GET /swap/health` returns the current status of the swap service, including the router integration and number of available trading pairs.

```
GET /swap/health
```

## Response

### 200 - Healthy

| Field | Type | Description |
|-------|------|-------------|
| `status` | `string` | Service status (`"ok"`) |
| `serverWallet` | `string` | Address of the swap server wallet |
| `bnbBalance` | `string` | BNB balance of the server wallet |
| `chainId` | `number` | Chain ID the swap service is connected to |
| `router` | `string` | DEX router in use (`"PancakeSwap V3"`) |
| `version` | `string` | Swap service version |

## Examples

### cURL

```bash
curl https://facilitator.b402.ai/swap/health
```

### TypeScript

```typescript
const response = await fetch('https://facilitator.b402.ai/swap/health');
const health = await response.json();

console.log('Status:', health.status);       // "ok"
console.log('Router:', health.router);       // "PancakeSwap V3"
console.log('Chain ID:', health.chainId);    // 56
console.log('Version:', health.version);
```

## Notes

- No authentication required
- Use this endpoint to verify the swap service is operational before initiating swaps
- A non-200 response or missing `status: "ok"` indicates the swap service is degraded
