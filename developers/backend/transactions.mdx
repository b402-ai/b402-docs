---
title: 'Transactions'
description: 'Query paginated transaction history'
---

`GET /transactions` returns paginated transaction history with optional filters. No authentication required. Responses are cached for **2 minutes**.

```
GET /transactions
```

## Query Parameters

All parameters are optional.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `txHash` | `string` | - | Transaction hash (0x-prefixed, 64 hex characters) |
| `from` | `string` | - | Buyer address (0x-prefixed, 40 hex characters) |
| `to` | `string` | - | Seller address (0x-prefixed, 40 hex characters) |
| `token` | `string` | - | Token contract address (0x-prefixed, 40 hex characters) |
| `startAt` | `string` | - | Start date (ISO 8601 or `YYYY-MM-DD`) |
| `endAt` | `string` | - | End date (ISO 8601 or `YYYY-MM-DD`) |
| `skip` | `string` | `"0"` | Number of records to skip |
| `take` | `string` | `"12"` | Number of records to return (max 1000) |

### Date Range Validation

- Maximum date range is **7 days**. Requests exceeding this return `400`.
- `endAt` must be greater than or equal to `startAt`.

## Response

### 200 - Transaction List

Results are ordered by `timestamp` descending (newest first).

**`data[]` fields:**

| Field | Type | Description |
|-------|------|-------------|
| `txHash` | `string` | On-chain transaction hash |
| `from` | `string` | Buyer wallet address |
| `to` | `string` | Seller wallet address |
| `token` | `string` | Token contract address |
| `amount` | `number` | USD amount (normalized by token decimals, multiplied by current price) |
| `timestamp` | `string` | ISO 8601 timestamp |

**`meta` fields:**

| Field | Type | Description |
|-------|------|-------------|
| `total` | `number` | Total matching transactions |
| `skip` | `number` | Records skipped |
| `take` | `number` | Records returned |

```json
{
  "data": [
    {
      "txHash": "0xabc123def456789012345678901234567890123456789012345678901234abcd",
      "from": "0xAbC1234567890aBcDeF1234567890AbCdEf123456",

      "token": "0x55d398326f99059fF775485246999027B3197955",

      "timestamp": "2024-03-01T12:00:00.000Z"

```

## Examples

### cURL

```bash
# Recent transactions (default pagination)
curl "https://backend.b402.ai/transactions"

# Filter by buyer address
curl "https://backend.b402.ai/transactions?from=0xAbC1234567890aBcDeF1234567890AbCdEf123456"

# Filter by date range with pagination
curl "https://backend.b402.ai/transactions?startAt=2024-03-01&endAt=2024-03-07&skip=0&take=50"

# Look up a specific transaction
curl "https://backend.b402.ai/transactions?txHash=0xabc123def456789012345678901234567890123456789012345678901234abcd"
```

### TypeScript

```typescript
const params = new URLSearchParams({
  from: '0xAbC1234567890aBcDeF1234567890AbCdEf123456',
  startAt: '2024-03-01',
  endAt: '2024-03-07',
  skip: '0',
  take: '50',
});

const response = await fetch(
  `https://backend.b402.ai/transactions?${params}`
);
const { data: transactions, meta } = await response.json();

console.log(`Showing ${transactions.length} of ${meta.total} transactions`);

for (const tx of transactions) {
  console.log(`${tx.txHash} | ${tx.from} -> ${tx.to} | $${tx.amount}`);
}
```

## Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `validation_error` | 400 | Invalid address format, hash format, date format, date range exceeds 7 days, or `endAt` before `startAt` |
