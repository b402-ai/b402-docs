---
title: 'Seller Leaderboard'
description: 'Paginated seller rankings by transaction count'
---

`GET /metrics/sellers` returns a paginated leaderboard of sellers ranked by transaction count (descending). No authentication required. Responses are cached for **2 minutes**.

```
GET /metrics/sellers
```

## Query Parameters

| Parameter | Type | Default | Max | Description |
|-----------|------|---------|-----|-------------|
| `skip` | `string` | `"0"` | - | Number of records to skip |
| `take` | `string` | `"12"` | 1000 | Number of records to return |

## Response

### 200 - Seller Leaderboard

**`data[]` fields:**

| Field | Type | Description |
|-------|------|-------------|
| `address` | `string` | Seller wallet address |
| `transactions` | `number` | Total transaction count |
| `volume` | `number` | Aggregate USD volume |
| `buyers` | `number` | Unique buyers transacted with |
| `latest` | `string` | ISO 8601 timestamp of most recent transaction |

**`meta` fields:**

| Field | Type | Description |
|-------|------|-------------|
| `total` | `number` | Total number of sellers |
| `skip` | `number` | Records skipped |
| `take` | `number` | Records returned |

```json
{
  "data": [
    {
      "address": "0x9876543210FeDcBa9876543210FeDcBa98765432",
      "transactions": 89,
      "volume": 8900.00,
      "buyers": 45,
      "latest": "2024-03-01T10:30:00.000Z"
    }
  ],
  "meta": {
    "total": 89,
    "skip": 0,
    "take": 12
  }
}
```

## Examples

### cURL

```bash
# First page (default)
curl "https://backend.b402.ai/metrics/sellers"

# Second page of 20
curl "https://backend.b402.ai/metrics/sellers?skip=20&take=20"
```

### TypeScript

```typescript
const params = new URLSearchParams({ skip: '0', take: '20' });
const response = await fetch(
  `https://backend.b402.ai/metrics/sellers?${params}`
);
const { data: sellers, meta } = await response.json();

console.log(`Showing ${sellers.length} of ${meta.total} sellers`);

for (const seller of sellers) {
  console.log(`${seller.address}: ${seller.transactions} txns, $${seller.volume}`);
}
```

## Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `validation_error` | 400 | Invalid `skip` or `take` values (e.g., `take` exceeds 1000) |
