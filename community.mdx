## Support

1. **Check the [FAQ](/resources/faq)** — Common questions about the protocol, pricing, and security
2. **Search the docs** — Use the search bar to find specific topics
3. **Join [Discord](https://discord.com/invite/b402)** — For real-time help from the community

## Contributing

b402 is an open protocol. You can contribute by:

- **Building SDKs** — Port the protocol to new languages (Go, Rust, Python)
- **Writing integrations** — Connect b402 to new frameworks and platforms
- **Improving docs** — Fix typos, add examples, or write guides

See the [GitHub organization](https://github.com/b402-ai) for contribution guidelines.
