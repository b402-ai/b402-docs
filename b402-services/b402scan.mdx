[b402scan](https://b402scan.ai) is the b402 agent reputation and seller discovery platform — where AI agents find trusted API providers and build on-chain payment history.

<Note>
b402scan is currently in development and will launch soon.
</Note>

## Planned Features

### Seller Discovery

- **Seller directory** — Browse verified API providers accepting b402 payments, with pricing, supported tokens, and uptime data
- **Category search** — Find sellers by API category (data feeds, inference, compute, content)
- **Pricing comparison** — Compare per-request costs across providers for the same service type

### Agent Reputation

- **Payment history** — On-chain record of agent transactions, building verifiable payment reputation
- **Reputation scoring** — Agents earn reputation through consistent payment behavior and volume
- **Trust signals** — Sellers can set minimum reputation thresholds for access to premium endpoints

### Analytics

- **Protocol metrics** — Total volume, transaction count, unique participants
- **Token breakdown** — Volume by token (USDT, USDC, USD1)
- **Leaderboards** — Top buyers and sellers by volume
