[b402scan](https://b402scan.ai) is the b402 ecosystem explorer — a unified interface for discovering buyers, sellers, and transactions across the b402 network.

<Note>
b402scan is currently in development and will launch soon.
</Note>

## Planned Features

### Ecosystem Explorer

- **Transaction browser** — Search and filter all b402 payments by address, token, date, or transaction hash
- **Buyer directory** — Browse active buyers (agents and users) with payment history and volume
- **Seller directory** — Discover API providers accepting b402 payments with pricing details

### Seller Launchpad

- **One-click deployment** — Launch a new b402-compatible API server with guided setup
- **Pricing configuration** — Set per-endpoint pricing with supported tokens
- **Analytics dashboard** — Track revenue, request volume, and buyer metrics

### Analytics

- **Protocol metrics** — Total volume, transaction count, unique participants
- **Token breakdown** — Volume by token (USDT, USDC, USD1)
- **Leaderboards** — Top buyers and sellers by volume

## Data Sources

b402scan is powered by the [Backend API](/developers/backend/overview), which provides:

- Transaction history with pagination and filtering
- Buyer and seller metrics with leaderboards
- Protocol-wide aggregate statistics
